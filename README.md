# Bio-Epigenetic Analyzer

A Python-based tool designed to analyze the intersection of genetic data and lifestyle factors. This project explores how daily habits might influence gene expression through a custom data engine.

## 🧬 Project Structure
* `main.py`: The entry point of the application.
* `dna_engine.py`: Core logic for processing genetic sequences and markers.
* `lifestyle_logic.py`: Handles the analysis of lifestyle factors (diet, sleep, etc.).
* `app_ui.py`: Manages the user interface components.
